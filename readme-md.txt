# Store Survey Analysis App

A Streamlit application for analyzing store survey results.

## Features

- Natural language queries for survey data
- Visualizations of store compliance
- Analysis of comments and feedback

## How to Use

1. Ask questions like:
   - "How many stores have not been visited?"
   - "Which stores didn't have a flyer?"
   - "Summarize the comments about the flyer"

2. View the visualized results and insights

## Deployment

This app is deployed on Streamlit Community Cloud.
